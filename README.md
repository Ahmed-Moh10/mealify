# 🍽️ Mealify Website

A modern and fully responsive restaurant landing page built using **HTML & CSS** with interactive features like **Dark Mode**, smooth navigation, and responsive design across all devices.

## 📌 Features

* ✅ Fully Responsive Design (Mobile / Tablet / Desktop)
* 🌙 Dark Mode Toggle (with icon switching 🌙/☀️)
* 📱 Mobile Navigation with Smooth Dropdown
* 🎯 Active Navbar Links (click + scroll tracking)
* 🎨 Smooth Animations & Hover Effects
* 🧑‍🍳 Chefs Section with Interactive Social Icons
* 🖼️ Gallery with Overlay Effects

* ⚡ Clean and Organized Code

## 🎯 Key Implementations

### 🔹 Responsive Design

Implemented using custom breakpoints:

* `< 640px` (Mobile)
* `640px - 767px` (Small devices)
* `768px - 1023px` (Tablet)
* `1024px+` (Desktop)

---

### 🔹 Dark Mode

* Toggle between Light/Dark mode
* Icon switches dynamically (Moon ↔ Sun)
* Saved using `localStorage`

---

### 🔹 Active Navigation

* Changes on click
* Updates automatically on scroll
* Works on all screen sizes including mobile dropdown
